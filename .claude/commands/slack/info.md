Show Slack channel details.

## Usage

```bash
hu slack info "#general"        # By name
hu slack info C12345678         # By ID
hu slack info "#general" -j     # JSON output
```

## Arguments

| Arg | Description |
|-----|-------------|
| `CHANNEL` | Channel name or ID (required) |

## Options

| Flag | Description |
|------|-------------|
| `-j, --json` | Output as JSON |
