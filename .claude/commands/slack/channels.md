List channels in the Slack workspace.

## Usage

```bash
hu slack channels           # Table output
hu slack channels -j        # JSON output
```

## Options

| Flag | Description |
|------|-------------|
| `-j, --json` | Output as JSON |

## Related Commands

| Command | Purpose |
|---------|---------|
| `hu slack info <CHANNEL>` | Channel details |
| `hu slack history <CHANNEL>` | Message history |
