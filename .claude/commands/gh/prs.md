List open pull requests authored by you.

## Usage

```bash
hu gh prs
```

## Default Action

List your open PRs in the current repo:

```bash
hu gh prs
```

## Output

For each PR:
- PR number and title
- State (open/draft/merged/closed)
- Branch name
- CI status

## Related Commands

| Command | Purpose |
|---------|---------|
| `hu gh failures` | Extract test failures from CI |
