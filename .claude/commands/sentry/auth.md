Set Sentry auth token.

## Usage

```bash
hu sentry auth -o myorg <TOKEN>
```

## Arguments

| Arg | Description |
|-----|-------------|
| `TOKEN` | Auth token (required) |

## Options

| Flag | Description |
|------|-------------|
| `-o, --org` | Organization slug (required) |

Token stored in `~/.config/hu/credentials.toml`.
