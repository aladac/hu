Show Sentry issue details.

## Usage

```bash
hu sentry show 12345
hu sentry show PROJ-ABC --json
```

## Arguments

| Arg | Description |
|-----|-------------|
| `ISSUE` | Issue ID or short ID (required) |

## Options

| Flag | Description |
|------|-------------|
| `--json` | Output as JSON |

## Related Commands

| Command | Purpose |
|---------|---------|
| `hu sentry events <ISSUE>` | Events for this issue |
| `hu sentry issues` | List all issues |
