Set NewRelic API key and account ID.

## Usage

```bash
hu newrelic auth -a 1234567 NRAK-...
```

## Arguments

| Arg | Description |
|-----|-------------|
| `KEY` | API key, NRAK-... (required) |

## Options

| Flag | Description |
|------|-------------|
| `-a, --account` | Account ID (required) |

Credentials stored in `~/.config/hu/credentials.toml`.
