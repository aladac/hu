Show all issues in the current sprint.

## Usage

```bash
hu jira sprint
```

## Default Action

List every issue in the active sprint (all assignees):

```bash
hu jira sprint
```

## Output

For each issue:
- Key, summary, status, assignee, priority

## Related Commands

| Command | Purpose |
|---------|---------|
| `hu jira tickets` | Only my tickets |
| `hu jira show <KEY>` | Ticket details |
