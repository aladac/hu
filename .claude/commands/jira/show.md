Show ticket details.

## Usage

```bash
hu jira show PROJ-123
```

## Arguments

| Arg | Description |
|-----|-------------|
| `KEY` | Ticket key, e.g., PROJ-123 (required) |

## Output

- Summary/title
- Status, priority, type
- Assignee, reporter
- Description
- Comments

## Related Commands

| Command | Purpose |
|---------|---------|
| `hu jira update <KEY>` | Update the ticket |
| `hu jira tickets` | List my tickets |
