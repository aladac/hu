List my tickets in current sprint.

## Usage

```bash
hu jira tickets
```

## Default Action

Show tickets assigned to you in the active sprint:

```bash
hu jira tickets
```

## Output

For each ticket:
- Key (e.g., PROJ-123)
- Summary/title
- Status
- Priority

## Related Commands

| Command | Purpose |
|---------|---------|
| `hu jira sprint` | All sprint issues (not just mine) |
| `hu jira show <KEY>` | Ticket details |
| `hu jira search <JQL>` | Custom search |
